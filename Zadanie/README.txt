Aby skorzystać ze skryptu, należy mieć zainstalowanego pythona oraz bibliotekę 'requests'

Bibliotekę 'requests' można zainstalować komendą "sudo pip install requests".

Jeżeli pip nie jest zainstalowany na maszynie można go zainstalować używając komendy
"python -m ensurepip --upgrade"

------------------------------------------------------------------------------------------

Aby poprawnie użyć skryptu, należy:

1)	
	Wpisać do pliku "nr.txt" numery komiksów 
	(jedna linia - jeden numer, w przypadku
	linii niezawierających liczb całkowitych
	program informuje o ich istnieniu i je
	zwyczajnie pomija)

2)		
	Uruchomić skrypt komendą "python skrypt.py"

3)	
	Otworzyć plik index.html
